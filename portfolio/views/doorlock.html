프로젝트 소개

NUCLEO-F401RE 보드 사용

디지털 도어락 장치를 구현하여 온습도 및 도어락 장치를 구현하였으며 조이스틱을 이용하여 비밀번호 설정이 가능하다.

주요 기능

기능 1

비밀번호를 올바르게 넣었을 경우 문이 성공적으로 열리면서 초록색

LED와 특정 음악이 재생되지만 틀렸을 경우 붉은색 LED와 다른 노래가 재생된다.

기능 2

문이 열린 상태에서 제한 시간을 경과하면 자동으로 문이 닫힌다.

기능 3

온습도 센서에서 온도와 습도 값을 읽어와 OLED Display에 출력한다.
